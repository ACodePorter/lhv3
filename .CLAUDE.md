# 量化交易系统 (Quant Trading System)

## 项目概述

这是一个集策略开发、回测分析、数据可视化于一体的量化交易平台，支持多种交易策略的配置、回测和参数优化。

## 技术栈

### 前端
- **框架**: React 18 + TypeScript
- **UI组件**: Ant Design 5.x
- **图表库**: ECharts 5.x
- **构建工具**: Vite 4.x
- **代码编辑器**: Monaco Editor / CodeMirror

### 后端
- **框架**: Python 3.8+ / FastAPI
- **数据处理**: pandas, numpy, scipy
- **技术分析**: TA-Lib
- **数据库**: SQLAlchemy + SQLite (backtesting.db)
- **数据源**: AKShare (A股), yfinance (Yahoo Finance)

## 核心功能模块

### 1. 策略系统
- **策略基类**: `StrategyBase` / `StrategyTemplate`
- **内置策略**:
  - `enhanced_ma_strategy.py` - 增强型均线策略
  - `extremum_strategy_v8.py` - 极值策略
  - `dividend_ma_t_strategy.py` - 分红均线策略
  - `moving_average.py` - 简单均线策略
  - `rsi_strategy.py` - RSI策略

### 2. 回测引擎
- 支持多种仓位模式: 固定比例、动态比例、分批建仓
- 绩效指标: 总收益率、年化收益率、夏普比率、最大回撤、胜率
- 支持手续费和滑点模拟

### 3. 参数优化
- 支持 Optuna、Hyperopt、scikit-optimize
- 并行参数扫描

### 4. 数据管理
- 多数据源支持 (AKShare, Yahoo Finance)
- 数据缓存机制
- 数据库持久化

## 数据库表结构

主要表:
- `strategies` - 策略定义
- `backtest_results` - 回测结果
- `strategy_parameter_spaces` - 参数空间定义
- `market_data` - 市场数据
- `stock_statistics` - 股票统计

## 开发规范

### 策略开发规范
1. 所有策略必须继承 `StrategyTemplate` 基类
2. 实现 `generate_signals()` 方法生成交易信号
3. 信号值: 1=买入, -1=卖出, 0=持有
4. **严禁使用未来数据** - 每个时间点只能使用该时间点及之前的数据
5. 策略代码存储在数据库中，通过数据库加载执行

### 代码规范
- Python 使用类型注解
- 函数添加文档字符串
- 日志使用 logging 模块
- 前端使用 TypeScript 严格模式

### API规范
- RESTful API设计
- 统一的错误响应格式
- 使用 Pydantic 进行数据验证

## 重要配置

### 后端配置 (`src/backend/config.py`)
```python
DATABASE_URL = "sqlite:///backtesting.db"
DEFAULT_COMMISSION_RATE = 0.0003  # 手续费率
DEFAULT_SLIPPAGE_RATE = 0.0001    # 滑点率
```

### 环境变量
- `DATABASE_URL` - 数据库连接
- `DEBUG` - 调试模式
- `QWEN_API_KEY` / `KIMI_API_KEY` / `DEEPSEEK_API_KEY` - AI API密钥

## 注意事项

1. **数据备份保护**: `data/backup/` 目录下的文件严禁修改
2. **策略更新**: 修改策略需要同步更新数据库中的策略代码
3. **参数同步**: 策略参数变更时需要同步更新 `strategy_parameter_spaces` 表
4. **临时脚本**: 使用后及时清理 `scripts/` 目录中的临时脚本

## 常用操作

### 添加新策略
1. 在 `src/backend/strategy/` 创建策略文件
2. 继承 `StrategyTemplate` 并实现 `generate_signals()`
3. 使用脚本将策略添加到数据库

### 运行回测
1. 通过前端界面选择策略和参数
2. 设置回测时间范围和初始资金
3. 执行回测并查看结果

### 参数优化
1. 在参数优化页面选择策略
2. 配置参数范围
3. 选择优化目标 (收益率、夏普比率等)
4. 运行优化并导入最优参数
